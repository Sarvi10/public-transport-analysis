# Passenger Journey Analysis

This project analyzes daily public transport passenger journeys across different service types. It includes historical trend analysis, daily growth rates, traffic index visualization, and 7-day passenger forecasts.

---

## Dataset

- **Source:** Local Excel file  
- **File:** `/content/Daily_Public_Transport_Passenger_Journeys_by_Service_Type_20250603.xlsx`  
- **Description:** Contains daily passenger counts for multiple public transport services including Local Route, Light Rail, Peak Service, Rapid Route, and School Service.
